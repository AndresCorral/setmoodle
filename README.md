## Preparar la instalación
### Sigue los pasos y estaras bien :ok_hand:


- sudo apt-get install git
- git clone https://github.com/AndresCorral/setmoodle.git
- cd setmoodle
- sudo chmod 777 setmoodle.sh
- ./setmoodle.sh
